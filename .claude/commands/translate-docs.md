以下の日本語ドキュメントを英語に翻訳して、指定のファイル名で保存してください。

- `CLAUDE.md` → `CLAUDE-en.md`
- `README.md` → `README-en.md`

翻訳のルール:
- 技術用語(cgo, goroutine, MIDI, CC, SysEx など)は英語のままにする
- コードブロック、ファイルパス、コマンド例はそのまま保持する
- Markdownの構造(見出しレベル、リスト、表)は完全に維持する
- 自然な技術英語で、冗長にならないように
- 既存の英語版ファイルがあれば、必要な箇所だけ差分更新してよい

翻訳後に `git diff --stat` で変更サマリを表示してください。
